# 💼 Job Analyzer

A data analysis and visualization project built using **Python, Pandas, Seaborn, Matplotlib, and Streamlit** to analyze job market trends, in-demand skills, and hiring patterns from job listing datasets.

---

## 🚀 Overview

The project focuses on:
- Cleaning and preprocessing job datasets  
- Performing exploratory data analysis (EDA)  
- Identifying in-demand technical skills  
- Applying feature engineering techniques  
- Visualizing hiring trends and skill distributions  
- Building an interactive Streamlit dashboard for data exploration  

---

## 🛠️ Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Streamlit  

---

## 📊 Features

- Data preprocessing and cleaning  
- Feature engineering on job datasets  
- Skill demand trend analysis  
- Interactive filtering and visualization  
- Streamlit dashboard for dynamic exploration  
- Graphical insights using Seaborn and Matplotlib
